# Cybersecurity Awareness Chatbot

## Overview

The Cybersecurity Awareness Chatbot is a desktop application developed using C#, WPF, and .NET 8.0. The purpose of the application is to help users learn about cybersecurity in an interactive and engaging way. Through a chat-based interface, users can ask questions and receive information about common online threats, safe internet practices, and cybersecurity concepts.

The chatbot was developed using Object-Oriented Programming (OOP) principles. Each class is responsible for a specific task, making the application easier to maintain, understand, and expand in the future. The system is designed to provide users with a simple and user-friendly experience while promoting cybersecurity awareness.

## Purpose of the Application

Cybersecurity has become an important concern in today's digital world. Many users are unaware of common online threats and how to protect themselves against them. This chatbot was created to:

* Educate users about cybersecurity threats and online safety.
* Provide practical advice on protecting personal information.
* Increase awareness of topics such as phishing scams, malware, passwords, privacy, and two-factor authentication.
* Offer guidance based on the user's questions and interests.
* Create an interactive learning experience through conversation.

## Key Features

### Chatbot Features

* Recognises cybersecurity-related keywords and topics.
* Provides different responses to keep conversations engaging.
* Identifies user emotions and responds appropriately.
* Remembers information such as the user's name and interests during the session.
* Supports follow-up questions and additional requests for information.

### User Interface Features

* Modern dark-themed interface for comfortable viewing.
* Chat bubbles that clearly separate user and chatbot messages.
* Message timestamps for better conversation tracking.
* Responsive layout that adjusts when the window size changes.
* Buttons for greetings, voice interaction, images, and clearing the chat.
* Placeholder text to guide users when entering messages.

### Additional Features

* Voice greeting functionality.
* Typing simulation for a more natural conversation experience.
* Automatic scrolling to the latest message.
* Support for sending messages using the Enter key.
* Chat history management.
* Built-in help system for user guidance.

## System Requirements

* Visual Studio 2022 or later
* .NET 8.0 or later
* Windows 10 or Windows 11
* Minimum 4 GB RAM
* At least 100 MB of free storage space
